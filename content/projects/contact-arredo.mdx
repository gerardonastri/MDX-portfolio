Contact Arredo is a sleek and modern website designed to showcase the high-quality furniture and interior design solutions offered by the company. The site is user-friendly, featuring a minimalist aesthetic with clean lines, elegant typography, and a neutral color palette that emphasizes professionalism and style.

## Features

1. **Stunning Homepage Design**  
   The homepage welcomes visitors with a striking hero section, featuring high-resolution images of furniture and interiors. It sets the tone for the brand with an elegant and modern aesthetic, accompanied by a clear call-to-action to explore the catalog or get in touch.
2. **Comprehensive Furniture Catalog**  
   A well-structured catalog showcases the full range of furniture and interior design solutions. Items are neatly categorized, each with detailed descriptions and high-quality images for easy browsing.
3. **About Us Section**  
   This section shares the companyâ€™s vision, mission, and expertise. It highlights the brand's commitment to quality craftsmanship, innovative designs, and customer satisfaction.
4. **Seamless Contact Options**  
   The contact page provides an intuitive form for users to reach out easily. Additional contact details, such as phone number and email address, are also available, ensuring smooth communication.
5. **Responsive and User-Friendly Design**  
   The website is fully responsive, guaranteeing a flawless user experience on desktops, tablets, and mobile devices. Navigation is effortless, with a layout optimized for all screen sizes.
6. **Fast Performance**  
   Powered by Vercel, the site delivers lightning-fast loading times and smooth transitions, ensuring a seamless browsing experience.
7. **Interactive and Engaging Elements**  
   Subtle animations and hover effects add a touch of interactivity to the design, enhancing user engagement while maintaining simplicity.

## Technologies

- **Typescript**: A syntactic superset of JavaScript which adds static typing
- **NextJs**: A React framework for building static and dynamic websites.
- **Vercel**: A cloud platform for static sites and serverless functions.
- **Tailwind CSS**: A utility-first CSS framework packed with classes
- **Framer Motion**: A tool for animations
- **ZOD**: A TypeScript-first schema declaration and validation library
- **Shadcn/UI**: A component distribution system
- **Nodemailer**: A Tool for sending email

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone https://github.com/gerardonastri/Contact-Arredo
cd hackfarm
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.

## Deployment

To deploy this project, you can use Vercel. You can link your GitHub repository
to Vercel and deploy the project with a single click.
