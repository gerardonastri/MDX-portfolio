## Features

- **OAuth**: Authentication with email or with google.
- **Session management**: Session management of the user with logout function
- **Searching**: Searching optimization with google places autocomplete
- **Geolocation**: User exact Geolocation (permissions will be asked)
- **Postgres database**: Postgres Database for storage.
- **Toast Notification**: Toast Notification when you submit a form.
- **Map directions**: drive route shown on map
- **Payment**: payment with Stripe

## Technologies

- **Expo**: React Native Framework
- **Neon**: Postgres database
- **NativeWind**: Tailwindcss for react native applications
- **Clerk**: User Management 
- **Expo secure store**: storing session values with expo
- **Geoapify**: Geolocation
- **Places api e directions api**: Google Apis for geolocations and directions
- **Zustand**: State management
- **Stripe**: secure payments

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone https://github.com/gerardonastri/uber-clone.git
cd uber-clone
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npx expo start
```


## Deployment

To deploy this project, you can use Expo to generate .apk or .aab file.
