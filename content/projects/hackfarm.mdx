HackFarm is a dynamic and modern platform designed for showcasing innovative tech projects and fostering a community of developers, tech enthusiasts, and entrepreneurs. The site combines an engaging design with functionality to provide a seamless experience for users exploring tech-related content.

## Features

1. **Engaging Homepage**  
   The homepage features an inviting hero section with eye-catching visuals, introducing visitors to HackFarm's mission and vision. Clear navigation options and call-to-actions lead users to explore projects or learn more about the community.
2. **Project Showcases**  
   A dedicated section displays featured tech projects, each with detailed descriptions, images, and key highlights. This feature is ideal for inspiring visitors and sharing cutting-edge innovations.
3. **Community-Oriented Content**  
   HackFarm emphasizes collaboration by presenting opportunities for visitors to connect with a community of like-minded individuals. This section promotes events, forums, or collaborative tech initiatives.
4. **Responsive and Optimized Design**  
   Built with responsiveness in mind, the website adapts flawlessly to any device, ensuring smooth navigation on desktops, tablets, and mobile screens.
5. **Fast and Reliable Performance**  
   Hosted on Vercel, HackFarm ensures quick loading times, smooth animations, and a hassle-free browsing experience for all visitors.
6. **Minimalist and Modern Aesthetic**  
   The site employs a sleek design language with clean typography, a dark-themed color palette, and interactive elements to create a professional yet approachable feel.
7. **Interactive and Functional UI**  
   Includes hover effects, animations, and an intuitive interface to keep users engaged and make navigation effortless.

## Technologies

- **Typescript**: A syntactic superset of JavaScript which adds static typing
- **NextJs**: A React framework for building static and dynamic websites.
- **Vercel**: A cloud platform for static sites and serverless functions.
- **Tailwind CSS**: A utility-first CSS framework packed with classes
- **Framer Motion**: A tool for animations
- **ZOD**: A TypeScript-first schema declaration and validation library
- **Shadcn/UI**: A component distribution system

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone https://github.com/gerardonastri/hackfarm.git
cd hackfarm
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.

## Deployment

To deploy this project, you can use Vercel. You can link your GitHub repository
to Vercel and deploy the project with a single click.
